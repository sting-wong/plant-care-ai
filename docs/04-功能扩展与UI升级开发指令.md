# 功能扩展 + UI 升级开发指令

## 背景

当前 MVP 功能已完成（拍照问诊、植物档案、养护提醒、成长记录）。AI API 暂不接入，使用 mock 数据。本次迭代目标是：对标 Planta/Greg 等竞品，补齐功能短板，升级 UI 到清新自然风格。

---

## 一、UI 全局升级（Planta 清新自然风）

### 设计语言定义

参考 Planta 的设计风格，整体方向是：自然、温暖、有呼吸感。

```
色彩系统：
- 主色：#2D7D46（深绿，用于按钮和强调）
- 辅色：#8BC34A（浅绿，用于背景和标签）
- 暖色点缀：#F5A623（琥珀色，用于提醒和施肥）
- 水色点缀：#4FC3F7（浅蓝，用于浇水相关）
- 背景色：#FAFDF7（极浅的绿白色，不是纯白）
- 卡片背景：#FFFFFF
- 文字主色：#1A2E1A（深绿黑，不是纯黑）
- 文字次色：#6B7B6B（灰绿）

圆角系统：
- 大卡片：20px
- 小卡片/按钮：12px
- 头像：圆形
- 输入框：12px

阴影：
- 卡片阴影：0 2px 12px rgba(45, 125, 70, 0.06)
- 不要用灰色阴影，用带绿色调的阴影

字体：
- 标题：18-20px，font-weight 700
- 正文：14px，font-weight 400
- 辅助文字：12px，font-weight 400

间距：
- 页面左右 padding：20px
- 卡片间距：16px
- 内容区域上下留白充足，有呼吸感

图标风格：
- 继续用 lucide-react，但搭配自然主题的装饰性插画
- 空状态用手绘风格的植物插画（可以用 SVG）
```

### 全局组件升级

**底部导航栏改造：**
- 当前 3 个 tab（首页/我的植物/提醒）→ 扩展为 5 个 tab
- 新增：发现（植物百科）、社区
- 图标用填充风格（选中态）vs 线条风格（未选中态）
- 选中态图标下方加一个小圆点指示器
- 导航栏背景加毛玻璃效果

```
底部导航结构：
[首页] [发现] [+] [社区] [我的]

- 首页：养护仪表盘
- 发现：植物百科 + 养护指南
- +：中间突出的圆形按钮（拍照/添加植物）
- 社区：动态流
- 我的：植物列表 + 设置
```

**顶部状态栏：**
- 去掉当前的黄色背景（那是浏览器书签栏）
- 页面顶部用渐变色 header（白色到极浅绿色渐变）

### 模块 1：植物百科 / 养护指南（发现页）

#### 页面路由
`/discover`

#### 功能描述

一个本地植物知识库，不依赖 AI API。用户可以浏览常见室内植物的养护指南。

#### 数据结构

```typescript
  difficulty: "easy" | "medium" | "hard";  // 养护难度
```

### 模块 2：光照计（环境工具）

#### 页面路由
`/tools/light-meter`（从发现页或植物详情页入口进入）

#### 功能描述

利用手机摄像头或环境光传感器，测量当前位置的光照强度，帮助用户判断植物摆放位置是否合适。

#### 实现方式

```typescript
// 方案 1：使用 AmbientLightSensor API（部分浏览器支持）
// 方案 2：使用摄像头画面亮度分析（更通用）

// 摄像头方案实现思路：
// 1. 打开前置/后置摄像头
// 2. 获取视频帧的平均亮度值
// 3. 将亮度值映射到光照等级

type LightLevel = "very_low" | "low" | "medium" | "bright" | "very_bright";

interface LightReading {
  level: LightLevel;
  lux: number;           // 估算的勒克斯值
  label: string;         // "散射光" | "明亮散射光" | "直射阳光" 等
  suitablePlants: string[]; // 适合这个光照的植物列表
  recommendation: string;   // "适合大多数观叶植物"
}
```

#### 页面结构

```

├─────────────────────────┤
│                         │
│     ┌─────────────┐     │

│     │  摄像头画面  │     │  ← 实时摄像头预览
│     │             │     │
│     └─────────────┘     │
│                         │
│     ┌─────────────┐     │
│     │   ☀️ 856 lux │     │  ← 实时数值显示
│     │  明亮散射光   │     │
│     └─────────────┘     │
│                         │
│  ━━━━━━━━━━━━━━━━━━━━━  │  ← 光照等级条
│  暗  低  中  亮  强      │
```
